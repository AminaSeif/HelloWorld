A Step-by-step CSS Detailed Description

CSS (Cascading Style Sheets) is a language that allows programmers to 'style' their websites. It is neither a programming language nor a markup language (for content).

CSS allows you to manipulate colors and visual effects on your website. Background and text colors, Italicizing, bolding, sizing and a lot of other visual efects are tweaked with CSS.

CSS can be run on your text editor, same as a HTML file. The extension has to be '.css' for it to work. To create the CSS code extension, follow these steps:

  1. Create a new folder in your HTML code file. Such a new folder is called a 'Directory'
  2. (For the purpose of this presentation) Name this new folder 'styles'
  3. Under this new folder create a file and name it 'styles.css'
  4. Write and Save your CSS code

Anatomy of the CSS link and How the invoking of CSS codes work

For a CSS code to run and affect a HTML code, it has to be invoked from the HTML code. It will be 'called upon' (invoked) by citing a relationship file; a relationship between the HTML and the CSS codes.Otherwise the CSS code will not work.

A relationship is cited as follows (within angle brackets): link href="style.css" rel="stylesheet", which basically tells the HTML code to proceed running the code it will find in the file 'style.css' inside the directory 'stylesheet'.

Notice that the 'Hyperreference link' ('href') that is used to invoke (call upon) the CSS code is written within the HTML code, but the actual CSS code will be in a different file, elsewhere

Let's dissect the CSS code for red paragraph text to understand how it works. The whole structure is called a ruleset. (which is often referred to as just rule.) Note the names of the individual parts:
  1. Selector
      2. This is the HTML element name at the start of the ruleset. It defines the element(s) to be styled (in this example, the elements considered were the paragraphs.). To style a different element, change the selector.
  2. Declaration
      3. This is a single rule like color: red;. It specifies which of the element's properties you want to style.
  3. Properties
      4. These are ways in which you can style an HTML element. (In this example, color is a property of the 'paragraph' elements.) In CSS, you choose which properties you want to affect in the rule.
  4. Property value
      5. To the right of the property—after the colon—there is the property value. This chooses one out of many possible appearances for a given property. (For example, there are many color values in addition to red.)

Note other important parts of the syntax, and extra functions that can be performed by CSS codes:

  • Apart from the selector, each ruleset must be wrapped in curly braces. ({})
  • Within each declaration, you must use a colon (:) to separate the property from its value or values.
  • Within each ruleset, you must use a semicolon (;) to separate each declaration from the next one.
  • To modify multiple property values in one ruleset, write them separated by semicolons, like this:

    p {

    color: red;

    width: 500px;

    border: 1px solid black;

    }

  • You can also select multiple elements and apply a single ruleset to all of them, Just Separate multiple selectors by commas.

    For example: p, li, h1 {

    color: red;

    }

  • You can also use different types of selectors.

    In HTML and CSS there are many types of selectors, which can make more specific selections

    Consider the examples bellow about selectors and what they select

    S/N Selector name What it selects Example
    1 Element selector (sometimes called a tag or type selector) All HTML elements of the specified type If the tag (in angle braces) of 'p' is selected/indicated, the code will select "Paragraphs'
    2 ID selector The element on the page with the specified ID On a given HTML page, each id value should be unique. #my-id selects (In angle braces)p id="my-id"(End of angle braces) or (In angle braces)a id="my-id" (End of angle braces)
    3 Class selector The element(s) on the page with the specified class Multiple instances of the same class can appear on a page. (Class is indicated by "(dot)name-of-class") For example: .my-class selects (In angle braces)p class="my-class"(End of angle braces) and (In angle braces)a class="my-class"(End of angle braces)
    4 Attribute selector The element(s) on the page with the specified attribute img[src] selects (In angle braces)img src="myimage.png"(End of angle braces) but not (In angle braces)img(End of angle braces)
    5 Pseudo-class selector The specified element(s), but only when in the specified state. (For example, when a cursor hovers over a link.) a:hover selects (In angle braces)a(End of angle braces), but only when the mouse pointer is hovering over the link.
